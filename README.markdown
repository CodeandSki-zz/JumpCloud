## Running the server
To run the server, make sure you have `node` and `npm` installed on your machine. Then do the following:
1. Pull down all the node dependencies:
  ```
  npm install
  ```
2. Run the server:
  ```
  node server.js
  ```
3. Point browser to:
  ```
  http://localhost:3004/
  ```
